Python

This image shows my rendition of the popular programming problem "Fizzbuzz", which is used as an entry level programming interview question to test the problem solving skills of fresh out of school programmers. The idea behind the game of "Fizzbuzz" is that is you start counting at 1, but if you reach a number that is divisible by 3, you say the word "Fizz", and if you reach a number divisible by 5, you would say "Buzz". If you have reached a number that is divisible by both 3 and 5, you would combine them, into the word "Fizzbuzz". This problem is popular because there are many different ways to solve it, each way saying something different about the person who is solving the problem.

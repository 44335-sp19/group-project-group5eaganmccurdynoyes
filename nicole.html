Python Project

I learned how to code Python in Programming 1 class at Northwest. Shown to the left is an output for a project that I have completed in Python code. The objective of this project was to create a random guessing game. First, I prompted the user to enter a maximum number for the range of the guessing game. I entered 100, so the game should generate a random number between 0 and 100. Then, the program asked me to give a number within the range. I happened to input 2, which was too low. The code told me that my guess was too low and asked me to enter another number. This kept happening until I guessed the correct number, 92, and the program told me that I was correct. Then, the program showed me how many guesses it took to get the number and asked me if I wanted to play again.
I used the random function (this had to be imported), input, nested while loops, and if-else statements, among other things, within the code to create this output. I liked creating this code because guessing games are fun and I was able to use great base-layer coding.

I showed one specific problem of code written out because it would be very difficult to include every file and every problem that I could work out. So, I used a database called Clearwater.
